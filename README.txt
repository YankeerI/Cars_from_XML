# Stax parsing of cars 

This is a Java project that provides a Stax parsing of Car objects from XML file.

## Technologies used
- Java
- XML

## Features
- Choosing a car by manufacturer/ by year of production/ by fuel type and consumption



## How to run it:
1. Clone the repository

2. Run the Spring Boot application  (the XML file is named cars.xml)
